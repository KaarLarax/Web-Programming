Aprendizajes como piezas

Tres parciales · una evolución

01

Parcial 1

Estructura y semántica

Aprendí a construir la base de una página: usar etiquetas con intención, organizar contenido, trabajar tablas y empezar a pensar en la web como documentos enlazados.

<table>
  <tr><td>contenido</td></tr>
</table>
  • HTML básico y estructura
  • Tablas, enlaces y recursos
  • Conceptos de internet y servidores web
Ver piezas del parcial 1
02

Parcial 2

Estilo, composición e interacción

Pasé de escribir contenido a diseñar experiencias: color, capas, gradientes, flexbox, grid y los primeros programas con decisiones, ciclos y eventos.

.layout {
  display: grid;
  gap: 1rem;
}
  • CSS externo y diseño responsive
  • Flexbox, grid, capas y efectos visuales
  • JavaScript con condicionales y ciclos
Ver piezas del parcial 2
03

Parcial 3

Navegador, componentes y servidor

Exploré el comportamiento del navegador con DOM y BOM, practiqué interfaces con Bootstrap y conecté formularios con PHP para entender el lado del servidor.

window.document
  .querySelector("#app")
  .classList.add("activo");
  • DOM, BOM, cookies y animación
  • Bootstrap con CDN, local y sin framework
  • PHP, formularios y CRUD
Ver piezas del parcial 3
